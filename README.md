# Siemens-NX-12-Title-Block-Revision-Table-Automation
• Designed a fully modular solution to automate drawing information entry in Siemens NX 12 engineering workflows using VB.Net Framework.                              • Integrated form-driven UI components with NXOpen APIs to auto-populate and manage title block fields and revision tables.                                          • Reduced manual data-entry errors significantly and improved engineering drawing turnaround time.
